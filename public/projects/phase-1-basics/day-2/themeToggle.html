Themeify

Experience the Switch

This simple application demonstrates a seamless theme toggle implementation using CSS Variables and Vanilla JavaScript.

🎨 Visual Consistency

By using CSS variables, we ensure that every component reacts instantly to theme changes without complex manual style updates.

💾 Persistence

Your preference is saved in local storage, so it stays the same even after you refresh the page or return later.

🌓 System Awareness

The app automatically detects your system's color scheme preference on the first visit for a personalized experience.
